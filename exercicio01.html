1º - Crie sequência para tomar banho

  1. Andar ate o banheiro
  2. Abrir a Porta do banheiro
  3. Fechar a porta do banheiro
  4. Ligara a luz
  5. Tirara a roupa
  6. Andar até o box
  7. Abrir o box
  8. Fechar o box
  9. Abrir o registro do chuveiro
  10. Verificar se a agua está quente
  11. entrar de baixo da agua
  12. pegar o sabonete / shampoo / condicionador
  13. passar o sabonete / shampoo /condicionador no corpo
  14. enxaguar-se
  15. pegar a toalha
  16. se enxugar
  17. banho finalzado

4º Trocar o Pneu de um carro.

  1. Parar o carro no acostamento
  2. Abrir a porta do carro
  3. fechar a porta do carro
  4. andar ate a mala e abrir a mala
  5. pegar o triangulo
  6. posicionar o triangulo no lugar certo
  7. após pegar o macaco
  8. posicionar o macaco no lugar certo
  9. levanatar o carro
  10. pegar a chave de roda
  11. retirar os 4 parafusos
  12. retirar roda com pneu furado
  13. ir ate a a mala e pegar o step
  14. posicinar o step no lugar certo
  15. aparafusar os 4 parafusos com a chave de roda
  16. abaixar o carro
  17. retirara o macaco
  18. pegar o macaco
  19. guardar o macaco no porta malas
  20. pegar o triangulo
  21. guardar o triangulo
  22. fechar a mala
  23. Fim

5º Como trocar a lampada

  1. Desligue o interruptor e espere a lâmpada esfriar.
  2. Use uma escada ou um tamborete para alcançar a lâmpada
  3. solte os parafusos se a lâmpada estiver dentro de uma luminária
  4. Use fita adesiva para remover a luminária
  5. tire a lâmpada velha do soquete.
  6. Compre uma nova lâmpada de mesma voltagem
  7. Reinstale a luminária

6º como fazer 1 bolo

  1. Selecionar tigela
  2. colocar farinha, de acordo com a medida
  3. selecionar ovos
  4. Colocar manteiga e açúcar a gosto
  5. Colocar leite
  6. Misturar todos os ingredientes na tigela
  7. Despejar a massa na forma
  8. Aguardar 40 minutos
  9. Retirar do forno
  10. Servir o bolo

Para que serve o Java

Java é uma linguagem de programação que permite a você implementar itens complexos em páginas web — toda vez que uma página da web faz mais do que simplesmente mostrar a você informação estática — mostrando conteúdo que se atualiza em um intervalo de tempo, mapas interativos ou gráficos 2D/3D animados, etc.

Para que serve o PHP

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor ) é uma linguagem de open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.

Para que serve o Java

Java é uma linguagem multiplataforma, orientada a objetos e centrada em rede que pode ser usada como uma plataforma em si. É uma linguagem de programação rápida, segura e confiável para codificar tudo, desde aplicações móveis e software empresarial até aplicações de big data e tecnologias do servidor.

Para que serve C#

O C# é uma linguagem multiplataforma. Sendo assim, você pode utilizá-la para desenvolver para plataformas web, dispositivos móveis e aplicações desktop. Com a praticidade dessa linguagem, você pode, de forma relativamente fácil, desenvolver desde projetos mais simples até projetos complexos e multiplataforma.

Para que serve C++

C++ é uma das linguagens mais versáteis que existem, permitindo desenvolver desde tarefas simples como aplicações na linha de comando ou web, até sistemas complexos de tempo real, muito usadas no mercado financeiro.
